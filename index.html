Создание карточек Anki с множественными вариантами ответа.

2017-06-28

Статья рассказывает о том, как добавить в Anki новый тип шаблона с множественными вариантами ответов.
Для реализации используется встроенный JavaScript рендерер карточек Anki, а также Firebase для синхронизации состояния между карточками вопроса и ответа.
